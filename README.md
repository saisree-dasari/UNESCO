# 🌍 Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites in Tableau

## 📖 Project Overview

Heritage Treasures is an interactive Tableau dashboard developed to analyze UNESCO World Heritage Sites worldwide. The project transforms raw heritage site data into meaningful visualizations that help users explore global heritage locations, identify trends, and gain insights through interactive dashboards.

The dashboard enables users to analyze World Heritage Sites by country, region, category, inscription year, and danger status using maps, charts, filters, and KPIs.

---

## 🎯 Objectives

- Analyze UNESCO World Heritage Sites across different countries.
- Visualize the distribution of Cultural, Natural, and Mixed heritage sites.
- Identify heritage sites that are classified as endangered.
- Study inscription trends over the years.
- Provide an interactive dashboard for better decision-making and analysis.
